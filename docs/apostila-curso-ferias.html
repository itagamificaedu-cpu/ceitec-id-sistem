📖 Teoria — Eletrônica para Quem Nunca Viu

O que é eletricidade? É o fluxo de elétrons (partículas minúsculas) por dentro de materiais condutores, como fios de cobre. Pense nela como água em um cano: a tensão (voltagem) é a pressão da água, a corrente é a quantidade que passa, e a resistência é o quanto o cano dificulta a passagem.

Componentes eletrônicos são peças que controlam esse fluxo. Cada um tem uma função específica, assim como válvulas, bombas e filtros em uma tubulação.

🚰 Analogia do cano: Tensão = pressão da bomba d'água · Corrente = litros por segundo · Resistência = tubo mais fino que dificulta a passagem

Protoboard é uma placa plástica com furos conectados internamente. Serve para montar circuitos temporários sem precisar soldar — encaixa, testa, ajusta. As colunas do meio são conectadas verticalmente; as fileiras das bordas (+ e –) são trilhos de energia horizontal.

🔧 Componentes do Dia

💡
LED
Light Emitting Diode — emite luz quando corrente passa. Tem polaridade: perna longa (+), perna curta (–)
〰️
RESISTOR
Limita a corrente. Sem ele o LED queima. As faixas coloridas indicam o valor em Ohms (Ω)
⚡
PROTOBOARD
Placa para montar circuitos sem solda. 830 furos. Ideal para testes e protótipos
🔋
BATERIA 9V
Fornece energia ao circuito. Tem terminal positivo (+) e negativo (–). Não inverta!
🔴
JUMPER
Cabos de conexão para a protoboard. Macho-Macho (M-M), Macho-Fêmea (M-F)
📏
MULTÍMETRO
Mede tensão, corrente e resistência. Essencial para diagnosticar problemas no circuito

🛠️ Atividade — Circuito LED na Protoboard

Diagrama do Circuito — LED + Resistor + Bateria 9V
1
Pegue a protoboard e os componentes
1 LED vermelho, 1 resistor 330Ω (faixas laranja-laranja-marrom), 2 jumpers, 1 bateria 9V com adaptador
2
Encaixe o LED na protoboard
Coloque a perna longa (+) na coluna E5 e a perna curta (–) na coluna E6. As pernas devem ficar em colunas diferentes.
3
Conecte o resistor em série
Uma perna do resistor na coluna D5 (mesma linha do LED +), outra perna na coluna A5. O resistor não tem polaridade — pode virar dos dois lados.
4
Ligue o trilho positivo ao resistor
Jumper do trilho + (vermelho) da protoboard até a coluna onde está a outra perna do resistor (A5).
5
Ligue o trilho negativo ao LED
Jumper do trilho – (preto) até a coluna da perna curta do LED (E6).
6
Conecte a bateria e confira o resultado
Plugue o adaptador da bateria 9V nos trilhos da protoboard (+vermelho, –preto). O LED deve acender! Se não acender, verifique a polaridade do LED.
🎯 MISSÃO DO DIA 1

Você fez um LED acender. Agora o desafio: faça um semáforo com 3 LEDs (vermelho, amarelo, verde) usando a mesma protoboard. Cada LED precisa de um resistor próprio.

  • Use 3 LEDs de cores diferentes
  • Cada LED precisa de um resistor 330Ω em série
  • Todos os negativos vão para o trilho GND
  • Todos os positivos ao trilho +5V
💡
Sabia que? O LED foi inventado em 1962 pelo físico Nick Holonyak Jr., mas o LED azul — que permite criar LEDs brancos e telas modernas — só foi criado em 1993. Os inventores do LED azul ganharam o Prêmio Nobel de Física em 2014. Cada vez que você vê uma tela de celular ou uma lâmpada LED, está usando essa tecnologia!

📖 O Que é o Arduino?

Arduino é uma placa com um microcontrolador — um computador minúsculo que executa um programa repetidamente. Ele tem pinos de entrada e saída (I/O) que permitem ler sensores e controlar componentes.

Você programa ele usando a IDE Arduino (software gratuito no computador). Escreve o código em C++ simplificado, clica em "Carregar" e ele começa a executar na placa imediatamente.

🧠 Analogia: O Arduino é o cérebro. Os sensores são os olhos e ouvidos. Os LEDs, motores e buzzers são os músculos. O código que você escreve é o que diz ao cérebro o que fazer com as informações dos sentidos.

Todo sketch Arduino tem dois blocos obrigatórios:
setup() — executado uma vez ao ligar (configurações iniciais)
loop() — executado para sempre em repetição (lógica principal)

💻 Seu Primeiro Código — LED Piscando

// LED Piscando — o "Hello World" do Arduino void setup() { pinMode(13, OUTPUT); // Pino 13 como saída } void loop() { digitalWrite(13, HIGH); // Liga o LED delay(1000); // Espera 1 segundo digitalWrite(13, LOW); // Desliga o LED delay(1000); // Espera 1 segundo }

🌡️ Sensor de Temperatura DHT11

🌡️
DHT11
Mede temperatura (0-50°C) e umidade (20-80%). 3 pinos: VCC (+), GND (–), DATA (sinal)
📡
HC-SR04
Sensor ultrassônico. Mede distância de 2cm a 400cm. 4 pinos: VCC, GND, TRIG, ECHO
🖥️
SERIAL MONITOR
Janela na IDE que mostra dados enviados pelo Arduino via USB. Use Serial.println() para imprimir
// Lendo distância com HC-SR04 const int trigPin = 9; const int echoPin = 10; void setup() { Serial.begin(9600); pinMode(trigPin, OUTPUT); pinMode(echoPin, INPUT); } void loop() { digitalWrite(trigPin, LOW); delayMicroseconds(2); digitalWrite(trigPin, HIGH); delayMicroseconds(10); digitalWrite(trigPin, LOW); long duracao = pulseIn(echoPin, HIGH); float distancia = duracao * 0.034 / 2; Serial.print("Distância: "); Serial.println(distancia); delay(500); }
🎯 MISSÃO DO DIA 2

Crie um alarme de proximidade: quando alguém se aproximar a menos de 20cm do sensor HC-SR04, o LED vermelho acende e o Serial Monitor exibe "⚠️ ALERTA!".

  • Quando distância < 20cm → LED vermelho ON + mensagem "ALERTA!"
  • Quando distância ≥ 20cm → LED verde ON + mensagem "OK"
  • Bônus: adicione um buzzer que apita quando alerta
🦇
Sabia que? O sensor HC-SR04 funciona como o eco dos morcegos! Ele emite um pulso ultrassônico (som acima da frequência que ouvimos) e mede quanto tempo leva para o eco retornar. Dividindo pela velocidade do som (343 m/s) e por 2 (ida e volta), você tem a distância exata.

📖 Motores e o Driver L298N

Motor DC gira quando recebe corrente elétrica. Inverta a polaridade e ele gira ao contrário. O problema: motores precisam de mais corrente do que o Arduino consegue fornecer (Arduino: 40mA por pino; motor: até 2A).

Por isso usamos o Driver L298N — uma ponte H. Ele recebe o sinal de controle (baixa corrente) do Arduino e usa energia de uma fonte separada (bateria) para alimentar os motores com a corrente necessária.

⚙️
MOTOR DC
Converte eletricidade em rotação. Com caixa de redução ganha torque. Sentido de rotação muda com a polaridade
🔌
DRIVER L298N
Ponte H dupla. Controla 2 motores DC. Pinos IN1-IN4 recebem sinal do Arduino. Pinos OUT1-OUT4 vão para os motores
🚗
CHASSI 2WD
Plataforma com 2 motores + 2 rodas traseiras + roda boba dianteira. Base do robô

🛠️ Montagem do Robô

1
Monte o chassi
Fixe os 2 motores DC nas laterais do chassi com os parafusos. Encaixe as rodas nos eixos dos motores. Parafuse a roda boba na frente.
2
Fixe o Arduino e o driver L298N
Use fita dupla face ou parafusos. Coloque o Arduino no centro-superior e o L298N atrás. Deixe os conectores acessíveis.
3
Conecte os motores ao L298N
Motor esquerdo → OUT1 e OUT2. Motor direito → OUT3 e OUT4. A polaridade define sentido — ajuste depois se necessário.
4
Conecte L298N ao Arduino
IN1→pino 2, IN2→pino 3, IN3→pino 4, IN4→pino 5. ENA e ENB em HIGH (ou ligue ao 5V para velocidade máxima).
5
Monte o sensor HC-SR04 na frente
Fixe o sensor apontando para frente com fita dupla face. TRIG→pino 9, ECHO→pino 10, VCC→5V, GND→GND.
6
Ligue as baterias e teste
Pack de 4 pilhas AA (6V) no L298N para os motores. Arduino alimentado pelo USB do computador durante programação.
// Robô autônomo — desvia de obstáculos const int IN1=2, IN2=3, IN3=4, IN4=5; const int TRIG=9, ECHO=10; float medirDistancia() { digitalWrite(TRIG, LOW); delayMicroseconds(2); digitalWrite(TRIG, HIGH); delayMicroseconds(10); digitalWrite(TRIG, LOW); return pulseIn(ECHO, HIGH) * 0.034 / 2; } void frente() { digitalWrite(IN1,HIGH); digitalWrite(IN2,LOW); digitalWrite(IN3,HIGH); digitalWrite(IN4,LOW); } void re() { digitalWrite(IN1,LOW); digitalWrite(IN2,HIGH); digitalWrite(IN3,LOW); digitalWrite(IN4,HIGH); } void direita(){ digitalWrite(IN1,LOW); digitalWrite(IN2,HIGH); digitalWrite(IN3,HIGH); digitalWrite(IN4,LOW); } void parar() { digitalWrite(IN1,LOW); digitalWrite(IN2,LOW); digitalWrite(IN3,LOW); digitalWrite(IN4,LOW); } void loop() { float dist = medirDistancia(); if (dist < 20) { // Obstáculo a menos de 20cm parar(); delay(300); re(); delay(500); direita(); delay(400); } else { frente(); } delay(100); }
🎯 MISSÃO DO DIA 3

Seu robô já desvia de obstáculos. Agora: programe uma dança de 10 segundos — uma sequência de movimentos que acontece quando o robô é ligado antes de entrar no modo autônomo.

  • Ao ligar: 10 segundos de coreografia (frente, ré, girar, etc.)
  • Depois dos 10 segundos: entra no modo autônomo normalmente
  • Bônus: adicione um LED que pisca durante a dança
🏭
Sabia que? O mesmo princípio do seu robô é usado em robôs industriais como o Roomba (aspirador robô) e em veículos autônomos da Tesla e Waymo. A diferença é que eles usam câmeras e LIDAR (radar a laser) em vez do HC-SR04. Mas a lógica fundamental — "detectou obstáculo → desvia" — é exatamente a mesma que você programou hoje!

📖 ESP32 — O Arduino com Superpoderes

ESP32 é um microcontrolador criado pela Espressif com WiFi e Bluetooth integrados. É mais poderoso que o Arduino Uno (dual core, 240MHz, 520KB RAM), mais barato e ainda tem muito mais pinos.

O mais incrível: ele pode funcionar como um servidor web — ou seja, criar um site acessível por qualquer celular na mesma rede WiFi. Você acessa pelo navegador do celular e controla o robô em tempo real!

Arduino Uno

CPU: 16 MHz · RAM: 2KB · Sem WiFi · ~R$45

ESP32 DevKit

CPU: 240 MHz · RAM: 520KB · WiFi+BT · ~R$35

💻 Servidor Web no ESP32

// Robô controlado por WiFi via ESP32 #include <WiFi.h> #include <WebServer.h> const char* ssid = "NomeDaSuaRede"; const char* password = "SenhaDaRede"; WebServer server(80); const int IN1=26, IN2=27, IN3=14, IN4=12; String pagina = "<!DOCTYPE html><html>" "<body style='background:#111;color:#fff;font-family:sans-serif;text-align:center'>" "<h1>🤖 Controle do Robô</h1>" "<a href='/frente'><button>▲ FRENTE</button></a><br>" "<a href='/re'><button>▼ RÉ</button></a>" "</body></html>"; void setup() { WiFi.begin(ssid, password); while (WiFi.status() != WL_CONNECTED) delay(500); server.on("/", [](){ server.send(200, "text/html", pagina); }); server.on("/frente",[](){ /* ligar motores frente */ server.sendHeader("Location","/"); server.send(303); }); server.begin(); } void loop() { server.handleClient(); }

Após carregar o código, abra o Serial Monitor e veja o IP do ESP32 (ex: 192.168.1.105). Digite esse endereço no navegador do celular (na mesma rede WiFi) e o painel de controle aparece!

GLOSSÁRIO MAKER

20 termos essenciais que todo maker precisa conhecer

ARDUINO
Plataforma de hardware e software de código aberto para criar projetos interativos. Fácil de programar e muito usada em protótipos.
ESP32
Microcontrolador com WiFi e Bluetooth integrados. Mais poderoso e barato que o Arduino Uno.
MICROCONTROLADOR
Computador completo em um único chip: processador, memória e I/O. O "cérebro" dos projetos eletrônicos.
PROTOBOARD
Placa para montar circuitos sem solda. Fácil para testes e ajustes rápidos.
GPIO
General Purpose Input/Output — pinos que podem ler sensores (entrada) ou acionar componentes (saída).
PWM
Pulse Width Modulation — técnica para simular saída analógica com sinal digital. Usado para controlar velocidade de motores e brilho de LEDs.
I2C / SPI
Protocolos de comunicação serial entre componentes. Permitem conectar vários dispositivos usando apenas 2 fios.
SENSOR
Componente que converte grandezas físicas (temperatura, distância, luz) em sinais elétricos que o microcontrolador pode ler.
ATUADOR
Componente que converte sinal elétrico em ação física. Exemplos: motor, servo, buzzer, display, relé.
DRIVER
Circuito intermediário entre o microcontrolador e um atuador que precisa de mais potência (ex: L298N para motores DC).
SKETCH
Nome dado ao programa escrito na IDE Arduino. Todo sketch tem as funções setup() e loop().
COMPILAR
Transformar o código que você escreveu em linguagem de máquina que o microcontrolador entende. A IDE faz isso automaticamente.
BIBLIOTECA
Conjunto de funções prontas para usar com um componente específico. Ex: biblioteca DHT para o sensor de temperatura.
SERIAL MONITOR
Ferramenta da IDE para comunicar com o Arduino via USB e ver dados em tempo real. Essencial para depuração.
IoT
Internet of Things (Internet das Coisas) — conectar objetos físicos à internet para monitorar e controlar remotamente.
MAKER
Cultura de criar, hackear e compartilhar. Makers acreditam que qualquer pessoa pode construir qualquer coisa com as ferramentas certas.
PROTOTIPAGEM
Processo de construir uma versão inicial de um produto para testar e aprimorar antes da versão final.
RESISTOR
Componente que limita a passagem de corrente elétrica. Medido em Ohms (Ω). Código de cores nas faixas indica o valor.
GND
Ground (terra) — referência de 0V do circuito. Todo componente precisa de uma conexão ao GND para funcionar.
OPEN SOURCE
Software ou hardware com código/projeto aberto para qualquer um usar, modificar e distribuir. Arduino é open source.
