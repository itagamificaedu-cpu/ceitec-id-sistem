Cabo USB no Arduino → porta USB do PC. O LED verde acende — Arduino ligado!
✓

Perna longa (+) à direita, perna curta (–) à esquerda. Deixe um furo entre elas.

Um lado na mesma fileira da perna longa do LED. O outro lado numa coluna livre. Resistor não tem polaridade.

Uma ponta no outro lado do resistor. A outra ponta no pino 13 do Arduino.
✓

Uma ponta na perna curta do LED. A outra ponta no pino GND do Arduino.
✓

Ferramentas → Placa → Arduino Uno. Depois Ferramentas → Porta → selecione a COM que aparecer.
✓
7
Digite o código e envie
Copie o código abaixo, apague o que tiver na tela e clique na seta (→) para enviar ao Arduino.
✓
💻 Código Arduino 
// Blink — LED piscando

void setup() {
  pinMode(13, OUTPUT); // Pino 13 = saída
}

void loop() {
  digitalWrite(13, HIGH); // Liga LED
  delay(500);             // Espera 0,5s
  digitalWrite(13, LOW);  // Desliga LED
  delay(500);             // Espera 0,5s
}
🎉

Funcionou? O LED pisca uma vez por segundo! Tente mudar 500 para 100 e veja o que acontece!

SEÇÃO 04
Como Funciona? 🧠

📌 O que o código faz

O Arduino executa em dois momentos:

setup() — roda uma vez quando liga. Define o pino 13 como saída (OUTPUT).

loop() — roda para sempre. Liga → espera → desliga → espera → recomeça... infinito!

🤔 Por que o resistor?

O LED aguenta no máximo 20mA. O Arduino manda muito mais. Sem resistor, é como abrir uma torneira numa xícara — transborda e queima. O resistor mantém a corrente no nível certo.

⚡ Os 3 fundamentos

Voltagem (V) = "pressão" da corrente (Arduino = 5V)
Corrente (A) = quantidade de elétrons que passam
Resistência (Ω) = "freio" que controla a corrente

Lei de Ohm: V = R × I

⭐ DESAFIO EXTRA
Faça 3 LEDs piscarem em sequência, como um semáforo!

💡 Dica do Desafio

Use 3 LEDs, 3 resistores e os pinos 11, 12 e 13. No setup() declare os 3 pinos como OUTPUT. No loop() use delay() entre cada LED para criar a sequência!
